## 初期設定

ホストにマウントしたディレクトリの所有者が `root` になってしまって、 `fluentd` が目的のログファイルをテイルできない。

```
[client] $ docker-machine ssh <host>
[host] $ sudo chown -R docker:1000 /var/log/fluentd/
[host] $ sudo chown -R docker:1000 /var/log/nginx/
```

## 起動

```
$ docker-compose up -d
```
